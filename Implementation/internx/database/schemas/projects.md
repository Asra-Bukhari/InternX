# Projects Collection

## Description
Stores projects posted by businesses.

## Fields

- businessId: Owner of project
- title: Project title
- description: Project details
- skillsRequired: Required skills
- difficulty: easy | medium | hard
- contractType: fixed | hourly
- applicants: List of student IDs (max 10)
- selectedStudent: Final selected student
- status: open | in-progress | completed
- createdAt: Timestamp
