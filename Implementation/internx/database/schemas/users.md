# Users Collection

## Description
Stores all users of the platform including students, businesses, and admins.

## Fields

- name: Full name of user
- email: Unique email address
- password: Encrypted password
- role: student | business | admin
- isVerified: Whether account is verified
- createdAt: Account creation time
- updatedAt: Last update time

## Example Users

### Student
- Name: Asra Bukhari
- Email: asra@gmail.com
- Role: student
